## 批量拖拽-文件后缀名重命名工具 v1.0

### 功能

+ 支持拖拽文件夹/单个文件/多个文件到GUI界面进行批量重命名
+ 后缀名检测
  + 全中文/检测名单后缀名 -> 替换为默认后缀名
  + 部分中文后缀名 -> 去除中文
  + 无后缀名 -> 添加默认后缀名

+ 涉及修改的文件、未知类型文件会高亮显示
+ 支持自定义跳过的后缀名
